## フック運用

### 確認タイミング
- 各todo完了時に、この指針ファイルを手動で確認
- 進行状況と指針の照合を行い、報告する

### 確認項目
1. **作業範囲の遵守確認**
   - ファイル作成だけでなく、環境構築指示も含めているか
   - 対象ファイル・パスを明確に提示しているか

2. **ユーザー承認の徹底**
   - Phase完了時にユーザー承認を求めているか
   - 次工程に進む前に確認を得ているか

3. **詳細性の確保**
   - todoが抜けなく構成されているか
   - 疑問点をユーザーに確認しているか

4. **バックアップの適切性**
   - バックアップが適切に作成されているか
   - 日付付きで識別可能な形式になっているか
